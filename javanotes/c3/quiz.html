Question 1:

What is an algorithm?
An unambiguous process that terminates after a specified number of steps or as a result of a met condition or error.
Question 2:

Explain briefly what is meant by "pseudocode" and how is it useful in the development of algorithms.
Long-hand written instruction of the problem. It is syntactically less strict than actual code.
Question 3:

What is a block statement? How are block statements used in Java programs.
It a control structure that isolates a group of statements together. They are used in large conditional structures, subroutines and anywhere local variable scope is desired to be enforced.
Question 4:

What is the main difference between a while loop and a do..while loop?
In a do..while loop, the execution block is run once regardless of the while condition. Whereas, in the while loop, the execution block is only run if as the while condition starts and remains true.
Question 5:

What does it mean to prime a loop?
Creating a condition in which the loop can operate before it's conditions are evaluated.
Question 6:

Explain what is meant by an animation and how a computer displays an animation.
A sequence of images whose display, over a period of time, are changed (hidden or shown). A computer can move, resize, or hide an image to create an animated effect.

Question 1:

Briefly explain what is meant by the syntax and the semantics of a programming language. Give an example to illustrate the difference between a syntax error and a semantics error.

Syntax it is the structure of the language. Semantics is accurately solving the problem intended by the program.
A syntax ERROR could be: 
boolean booleanValue
because the syntax of the language specifies that a semi-colin end each line that is not a statement.

A semantic error could be:
integer halfOfSeven = 7/2;
because this is syntactically correct, however any values evaluated with this variable would likely be wrong since
7/2 is actually 3.5.
Question 2:

What does the computer do when it executes a variable declaration statement. Give an example.
It sets aside a portion of memory for use in storing a value, of a specified type, that is accessable by the declared name. integer aVariable; This will set aside 4 bytes (32 bits) of memory for use in storing a numeric value, from -2147483648 TO 2147483647, and can be assigned or access by the name aVariable.
Question 3:

What is a type, as this term relates to programming?
It indicates the space, in bytes, used to store a variable and the allowable values that can be stored in the memory location.
Question 4:

One of the primitive types in Java is boolean. What is the boolean type? Where are boolean values used? What are its possible values?
It is a logical true or false value. They can be used in conditional and loop expressions. Possible values that can be assigned as booleans are true and false. Expressions however also can evaluate to booleans. e.g. 1<0 ::= (boolean)false
Question 5:

Give the meaning of each of the following Java operators:

						a)    ++

						b)    &&

						c)    !=
						
	a) increment operator. This will increment the value of a variable by one. if it is used in an assignment statement,
	it's operation depends on where it appers in relation to the variable it is operating on.

	b) boolean and operator. This will return a boolean value based on the expressions surrounding it. it will return
	true only if both expressions are true, otherwise it will yield false.

	c) relational not equal to operator. This will test that the surrounding expressions are not equal to one another.
Question 6:

Explain what is meant by an assignment statement, and give an example. What are assignment statements used for?
Assign a value to the memory location accessed through the name of the variable being used. aVariable = aValue; The aVariable is the variable name, that references a memory location. This assignment statement is putting the value (aValue) in this memory location. They are used to store values in a program.
Question 7:

What is meant by precedence of operators?
The order in which operators are evaluated in a program. e.g. 5 - 2 * 2 == 1 // NOT == 6 because * is evaluated before -
Question 8:

What is a literal?
The characters that you type to represent a value. e.g. 10L and (long)10 are the same value with different literals used to represent this value.
Question 9:

In Java, classes have two fundamentally different purposes. What are they?
Defining the available attributes and subroutines used in the program and creating the program.
Question 10:

What is the difference between the statement "x = TextIO.getDouble();" and the statement "x = TextIO.getlnDouble();"

The first statement will read successive double values from the user while the second will read the first double and discard the remainder of the line includeing the newline.
Question 11:

Explain why the value of the expression 2 + 3 + "test" is the string "5test" while the value of the expression "test" + 2 + 3 is the string "test23". What is the value of "test" + 2 * 3 ?

The first two values are derived from the precedence of the + operator and the automatic type conversion done by the java compiler. 2+3 is evaluated first which is then converted to a string and concatenated to "test". While test is evaluated first as a string literal and 2 then 3 are converted into strings and concatenated to it. "test" + 2 * 3 == "test6"

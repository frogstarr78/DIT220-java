Quiz on Chapter 6

This page contains questions on Chapter 6 of Introduction to Programming Using Java. You should be able to answer these questions about studying that chapter. Sample answers to these questions can be found here.

Question 2:

Question 3:

Question 4:

Question 5:

Draw the picture that will be produced by the following paintComponent() method:

public static void paintComponent(Graphics g) {
    super.paintComponent(g);
    for (int i=10; i <= 210; i = i + 50)
       for (int j = 10; j <= 210; j = j + 50)
          g.drawLine(i,10,j,60);
}

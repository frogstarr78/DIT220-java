Introduction to Programming Using Java, Fifth Edition

Version 5.0, Part I, August 2006

Author:  David J. Eck  (eck@hws.edu)

Welcome to Part I of the upcoming fifth edition of Introduction to Programming Using Java. The remainder of the fifth edition is still under construction. Part I contains enough material for a one-semester college-level course on introductory programming. The fifth edition covers Java 5.0. Many of the examples and many of the applets that appear on the web site require Java 5.0 (or higher). The complete fourth edition can be found at http://math.hws.edu/javanotes/.

Introduction to Programming Using Java is a free, on-line textbook. It is directed mainly towards beginning programmers, although it might also be useful for experienced programmers who want to learn something about Java. It is certainly not meant to provide complete coverage of the Java language.

A PDF version of Part I is available; note that it does not include quiz answers, programming exercise solutions, or source code files. The PDF version can be downloaded from:

http://math.hws.edu/eck/cs124/downloads/javanotes5_part1.pdf

and a ZIP archive the web site can be downloaded from
